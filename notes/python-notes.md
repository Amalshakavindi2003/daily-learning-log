# Python Notes

## Variables
```python
name = "Amalsha"
age = 22
is_student = True
```

## Lists
```python
fruits = ["apple", "banana", "mango"]
fruits.append("orange")
fruits.remove("apple")
print(fruits[0])
```

## Dictionaries
```python
person = {
  "name": "Amalsha",
  "age": 22,
  "city": "Colombo"
}
print(person["name"])
```

## Functions
```python
def greet(name):
  return "Hello " + name

print(greet("Amalsha"))
```

## Loops
```python
for i in range(5):
  print(i)

while x > 0:
  x -= 1
```

## Useful Built-in Functions
- print() → display output
- input() → take user input
- len() → get length
- range() → generate numbers
- type() → check data type
- int(), str(), float() → type conversion

# HTML Tags Cheat Sheet

- <html> → root element
- <head> → document info
- <body> → page content
- <h1> to <h6> → headings
- <p> → paragraph
- <a> → link
- <img> → image
- <div> → container
- <ul> → unordered list
- <ol> → ordered list
- <li> → list item
- <form> → form element
- <input> → input field
- <button> → clickable button

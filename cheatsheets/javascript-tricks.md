# JavaScript Tricks Cheat Sheet

- console.log() → print to console
- typeof → check data type
- .length → get string or array length
- .toUpperCase() → convert to uppercase
- .toLowerCase() → convert to lowercase
- .trim() → remove spaces
- .split() → split string into array
- .join() → join array into string
- .includes() → check if value exists
- .indexOf() → find position of value
- Math.random() → generate random number
- Math.floor() → round down
- Math.max() → find maximum value
- Math.min() → find minimum value
- parseInt() → convert string to integer
- parseFloat() → convert string to float
- JSON.stringify() → convert to JSON
- JSON.parse() → parse JSON string
